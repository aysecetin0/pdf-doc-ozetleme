# pdf-doc-ozetleme
Flask tabanlı bu web uygulaması; kullanıcıların PDF formatındaki belgelerini sisteme yükleyerek içeriklerini hızlı ve etkili bir şekilde özetlemelerini sağlar. Yüklenen dokümanları ve oluşturulan özet geçmişini SQLite veritabanında saklayarak modern, dinamik ve kullanıcı dostu bir arayüz üzerinden yönetim imkanı sunar.
